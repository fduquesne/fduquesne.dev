Un outil existe pour créer des applications en VueJS simplement : Vue CLI. Pour pouvoir l'installer, il vous faudra installer [Node.js et npm](https://nodejs.org/en/) puis lancer la commande suivante :

```bash
npm install -g @vue/cli

# Pour vérifier que l'installation s'est bien effectuée
vue --version
```

## Initialiser le projet

Vue CLI précédemment installé permet d'initialiser un projet VueJS rapidement en lançant la commande suivante :

```bash
vue create electron-vue-application
# "electron-vue-application" est le nom de votre application.
# Vous pouvez le changer par le nom de votre application.
```

La commande vous demande alors de pick a preset. Je vous conseille de sélectionner _Manually select features_. Suite à celà, plusieurs features peuvent être appliquées à votre nouveau projet.
